# HR-Anti-sexual

# 🌻 TK3C 守護您 - 工作場所性騷擾防治專區

這是一個為 TK3C 打造的內部員工關懷與防治性騷擾宣導網頁。我們承諾提供一個安全、尊嚴、無歧視的工作環境，並透過此平台提供同仁明確的防治知識、自保指南以及申訴管道。

## ✨ 專案特色 (Features)

* **純前端輕量實作**：以單一 `index.html` 檔案構成，無須繁瑣的環境建置或依賴套件 (Zero Dependencies)。
* **響應式網頁設計 (RWD)**：完美適配桌上型電腦、平板及手機螢幕，不同裝置皆有最佳的閱讀體驗。
* **流暢的分頁切換**：使用原生 JavaScript (Vanilla JS) 打造流暢的頁籤切換效果。
* **互動式元件**：包含摺疊選單 (Accordion) 整理常見 Q&A，以及「一鍵複製」聯絡資訊功能，提升使用者體驗。

## 📑 核心內容區塊

本專區分為四個主要頁籤：

1.  **認識職場性騷擾**：定義性騷擾範疇、常見類型（利益交換型、敵意工作環境型）及構成要件。
2.  **如何防範職場性騷擾**：提供建立身體自主界線、保持公務互動專業等防範原則，並重申公司的零容忍政策。
3.  **應對方式與陪伴求助**：提供「堅定說不」與「尋求協助」的行動建議，並詳細列出 10 大自保紀錄關鍵重點。
4.  **申訴程序與救濟**：透明化申訴三大步驟，並提供專屬員工關懷熱線與信箱。

## 🛠️ 技術棧 (Tech Stack)

* **HTML5**: 語意化標籤與網頁結構。
* **CSS3**: Flexbox, CSS Grid 網格佈局，以及 `@media` 查詢實現 RWD。
* **JavaScript (ES5+)**: 控制 DOM 元素實現頁籤切換與複製到剪貼簿功能。

## 🚀 快速開始 (Getting Started)

由於此專案為純靜態網頁，您只需要將程式碼下載到本地端即可瀏覽，無須啟動任何伺服器。

1.  將此專案 Clone 到本地：
    ```bash
    git clone [https://github.com/您的帳號/您的專案名稱.git](https://github.com/您的帳號/您的專案名稱.git)
    ```
2.  在專案資料夾中，直接對著 `index.html` 點擊兩下，或是將其拖曳至任何現代瀏覽器（如 Chrome, Edge, Safari, Firefox）中開啟即可。
